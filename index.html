CoWIN Vaccine Availability Checker
    Derived from the official Government of India Portal Co-WIN. Beneficial for the ones who are struggling to find a vaccination center with an available slot in your district.
    If you find an available slot, immediately book the slots from the official portal for Co-WIN Registration.


    Features: Saving your state and district, removing unavailable slots, alert you with a GUITAR TONE on finding an available slot, the rest is same as the original Co-WIN app.
    Implementation: Created using the Co-WIN Public API. This tool is open source, you can find the code here at my git [link].
    Known bugs: The data might be inconsistent with the actual website, because the public API gives slightly older cached data.
    Feedback: If you find any bugs or have any suggestions to improve the tool, you may write to be in various social media @AxlHazarika
